# ContactUs_Web
Contact Us 웹 페이지 만들기

기능 구현

  - 화면단(JSP)와 Controller 연동 구현
  - Controller와 Service 연동 구현
  - Service(ServiceImpl)와 DAO(DAOImpl), DAO(DAOImpl)와 Mapper(.XML) 연동 구현
  - VO 객체 생성
  - MYSQL DB 데이터 저장
